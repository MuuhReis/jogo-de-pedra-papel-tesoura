# jogo-de-pedra-papel-tesoura
Projeto desenvolvido em Java que simula o clássico jogo de Pedra, Papel e Tesoura contra o computador, com interação via terminal.

⸻

Funcionalidades
	•	Menu interativo com opções:
	  •	Pedra 🗿
	  •	Papel 📃
	  •	Tesoura ✂️
	  •	Sair do jogo
	  •	Escolha do jogador via teclado
	  •	Geração de jogada aleatória do computador
	  •	Comparação automática das jogadas
	•	Exibição do resultado:
	  •	Vitória
	  •	Derrota
	  •	Empate

⸻

Tecnologias utilizadas
	•	Java
	•	Classe Scanner para entrada de dados
	•	Estruturas condicionais (if/else)
	•	Estrutura de repetição (while)
	•	Geração de números aleatórios com Math.random()

⸻

Aprendizados

Durante o desenvolvimento deste projeto, foram praticados:
	•	Lógica de jogos
	•	Comparação de condições múltiplas
	•	Uso de números aleatórios
	•	Interação com o usuário via terminal
	•	Estruturação de código

⸻

Exemplo de execução

O jogador escolhe uma opção e o computador responde com uma jogada aleatória, exibindo o resultado da rodada no terminal.
